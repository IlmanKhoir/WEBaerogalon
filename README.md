# Galon App - Aero Tripandawa

Aplikasi manajemen galon untuk Aero Tripandawa. Sistem ini dibangun menggunakan framework Laravel untuk memudahkan pengelolaan bisnis galon air.

## Fitur

- Manajemen pelanggan
- Pencatatan transaksi
- Tracking pengiriman
- Laporan penjualan
- Dashboard analytics

## Teknologi

- Laravel 10.x
- PHP 8.1+
- MySQL
- Bootstrap 5
- JavaScript/jQuery
